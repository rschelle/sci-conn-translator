What's This?

The Connotation Changer takes in text and a new connotation from the user and outputs modified text with the new connotation. Unlike other connotation changers we preserve the original meaning of the text.

How does this project work?

Our Connotation Changer uses the Vader Lexicon dataset. We used built in functions from the python standard library to extract the relevant data. After extraction, the data was cleaned with the regular expression library. Moreover, we processed the data using beautifulsoup4's webscraper to construct a python dictionary of synonyms from synonym.com. This dictionary of synonyms consists of keys, which correspond to words in the original text that we will change, and values for each key, which consist of a list of tuples containing synonyms and a sentiment difference between the word and synonym. When the user inputs text and a new connotation on the slider (represented as a sentiment increment), these are passed as inputs to our algorithm. Our algorithm is then used to find the synonym of the original word in the text with a sentiment difference closest to the sentiment increment. If no synonym is within a reasonable sentiment increment of the original word, then the original word is not replaced. Otherwise, we replace the word with the closest synonym based on the sentiment increment given. Once all words have been processed, we return the modified text.
