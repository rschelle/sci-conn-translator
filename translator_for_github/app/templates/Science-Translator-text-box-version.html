How It Works

Our AI model is Google's T5 Transformer model, used to generate the summaries of our scientific articles. T5 is a model that takes in a source text input and a target text output, meaning we input text into T5 that it will work with in order to generate the target text. We fed into the model a database of arxiv articles, an open repository of research papers, with an entire article (excluding the abstract) as the source text and its abstract as the target text.
